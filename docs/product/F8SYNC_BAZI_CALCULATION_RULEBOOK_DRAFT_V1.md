# F8SYNC BaZi Calculation Rulebook Draft V1

**Milestone:** 0B - Methodology Lock and Intelligence Decision Pack  
**Date:** 2026-06-17  
**Status:** Draft, not approved for implementation  

This rulebook draft defines structure and approval gates. It does not contain final BaZi formulas, source tables, solar-term values, Four Pillars expected outputs, or Golden Test expected values.

## Rule Status Key

| Status | Meaning |
|---|---|
| LOCKED | Explicit current directive. |
| PROPOSED | Recommended for V1 approval. |
| PENDING_PRODUCT_OWNER | KT decision required. |
| PENDING_EXPERT_VALIDATION | BaZi/calendar expert validation required. |
| DEFERRED_AFTER_V1 | Not part of first V1 implementation. |

## Core Restrictions

- Do not fabricate missing birth time.
- Do not use AI to calculate BaZi, element balance, timing, evidence, confidence, or archetypes.
- Do not fill Four Pillars expected values until the rulebook and reference results are approved.
- Do not treat current `mock-bazi-2026-06`, `mock-timing-2026-06`, or `identity.mapping.v8.0.0` behavior as approved methodology.
